# 🧙‍♂️ ホグワーツ・スペル・ガチャ (Hogwarts Spell Gacha)

ハリー・ポッター（HP-API）のデータを取り入れた、魔法界体験型のガチャ＆呪文デュエルWebアプリケーションです。  
ガチャで呪文を収集・図鑑をコンプリートし、実際に**自分の声（音声認識）で呪文を唱えて敵と戦う**ことができます！

---

## 🌟 主な機能

### 🎰 1. 魔法ガチャ（1連 / 10連）
* **HP-API** から取得したリアルな呪文データを使用。
* R / SR / SSR / LR の**4段階レアリティ演出**（豪華な発光・カード出現アニメーション）。
* 100ガリオン金貨で1回、1000ガリオン金貨で10連ガチャが回せます。

### 📖 2. 魔法図鑑（コレクション）
* ガチャで獲得した呪文が自動登録される図鑑機能。
* 呪文の名前や効果を確認可能。収集率（達成度）も一目でわかります。

### ⚔️ 3. 音声認識 呪文デュエル（ミニゲーム）
* プレイヤーvs闇の魔法使いの対戦バトル。
* **マイク機能（Web Speech API）**を活用し、**「図鑑に登録されている呪文」を声で唱えることで技を発動！**
* 正しく発唱できれば大ダメージを与えて撃破。未習得の呪文や発音失敗時は不発となり反撃を受けます。

### 🪄 4. 魔法体験アクション
* **組分け帽子**: 4つの寮（グリフィンドール、スリザリン、レイブンクロー、ハッフルパフ）から所属を選択可能。選択した寮によってアプリ全体のテーマカラーが変化。
* **杖の選択 & 魔法陣**: お気に入りの杖を選び、画面上の魔法陣をドラッグして詠唱体験。
* **日刊予言者新聞（ログインボーナス）**: アプリ起動時に新聞風ポップアップが表示され、ガチャに必要な「ガリオン金貨」を獲得可能。

---

## 🛠️ 技術構成

* **フロントエンド**: React (Vite / CRA)
* **スタイル**: CSS3 (CSS Variables, Keyframe Animations)
* **API**: [HP-API (Harry Potter API)](https://hp-api.onrender.com/)
* **ブラウザAPI**: Web Speech API (音声認識)

---

## 🚀 使い方 / プレイ手順

1. **アプリ起動**: 日刊予言者新聞からボーナス金貨を受け取ります。
2. **ガチャ**: ガチャ画面で「1連」または「10連」を回して呪文を解放します。
3. **図鑑確認**: 「魔法図鑑」タブでゲットした呪文（例: `Lumos`, `Alohomora` など）を確認します。
4. **デュエルで詠唱**: 「呪文デュエル」タブに移動し、マイクボタンを押してゲットした呪文の名前を声で唱えて敵を倒します！
